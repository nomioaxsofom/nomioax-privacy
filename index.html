Política de Privacidad

Esta Política de Privacidad se encuentra disponible públicamente en:
https://nomioaxsofom.github.io/nomioax-privacy/

1. Identidad del responsable

Nomioax es una aplicación operada por Fernando Domínguez Velásquez, a través de FAMValue Consultores, con operaciones en México.

Para cualquier asunto relacionado con esta Política de Privacidad o con el tratamiento de datos personales, el usuario puede contactar al responsable a través del siguiente correo electrónico:

contacto@nomioax.com

2. Datos personales que se recopilan

La aplicación Nomioax requiere el registro de usuario para su funcionamiento. Durante el uso de la aplicación, se pueden recopilar los siguientes datos personales:

  • Nombre completo
  • Dirección de correo electrónico
  • Número telefónico
  • Contraseña de acceso
  • Información bancaria
  • Información legal, fiscal o crediticia relacionada con los servicios ofrecidos

Nomioax no recopila datos de ubicación del dispositivo.

3. Datos sensibles y documentos

Nomioax puede tratar datos personales sensibles, incluyendo información financiera, bancaria, legal o crediticia, así como documentos que el usuario cargue voluntariamente a la plataforma.

  • Se utilizan exclusivamente para las finalidades del servicio.
  • Se almacenan dentro de la infraestructura tecnológica del sistema.
  • No se utilizan para fines distintos a los establecidos en esta Política.

4. Finalidad del tratamiento de los datos

  • Crear y administrar la cuenta del usuario.
  • Verificar la identidad del usuario.
  • Procesar solicitudes y operaciones.
  • Gestionar información bancaria y legal.
  • Enviar códigos de verificación y recuperación.
  • Proporcionar soporte técnico.
  • Cumplir obligaciones legales y regulatorias.
  • Mejorar funcionalidad, estabilidad y seguridad.

5. Uso de SMS y autenticación

Nomioax utiliza mensajes SMS únicamente para el envío de códigos de verificación, autenticación y recuperación de cuenta.

No se utilizan SMS con fines publicitarios ni de marketing.

6. Uso de servicios de terceros

Nomioax utiliza servicios de terceros como proveedores tecnológicos, incluyendo servicios de infraestructura en la nube proporcionados por Microsoft Azure.

7. Infraestructura tecnológica y tratamiento de datos

  • Automatización de procesos mediante Azure DevOps.
  • Servicios desplegados en Azure App Service.
  • Base de datos PostgreSQL administrada.

8. Seguridad de la información

Nomioax implementa medidas técnicas y organizativas razonables para proteger los datos personales, sin que ello implique una garantía absoluta.

9. Conservación de los datos

Los datos se conservarán únicamente durante el tiempo necesario para cumplir con las finalidades descritas o conforme a disposiciones legales aplicables.

10. Derechos del usuario

El usuario puede ejercer los derechos ARCO enviando una solicitud a: contacto@nomioax.com

11. Eliminación de cuenta y datos

La eliminación de cuenta y datos puede solicitarse por correo electrónico.

12. Público objetivo

Nomioax no está dirigida a menores de 13 años.

13. Cambios a esta Política

Cualquier cambio será publicado en esta misma URL.

14. Legislación aplicable

Esta política se rige por la Ley Federal de Protección de Datos Personales en Posesión de los Particulares.

15. Transferencias de datos personales

Destinatario Finalidad
Círculo de Crédito Consulta y reporte del historial crediticio.
Autoridades financieras y fiscales Cumplimiento de obligaciones legales y PLD/FT.
Microsoft Azure Infraestructura tecnológica y seguridad.
Despachos legales o de cobranza Gestión de recuperación de adeudos.

16. Uso de datos para fines secundarios

El usuario puede limitar o revocar el uso de datos para fines secundarios escribiendo a contacto@nomioax.com.

17. Uso de cookies y tecnologías de rastreo

Nomioax puede utilizar tecnologías de analítica para mejorar el servicio. No utiliza cookies ni publicidad personalizada.

18. Procedimiento para ejercer Derechos ARCO

  1. Nombre completo
  2. Medio de contacto
  3. Identificación oficial
  4. Descripción clara de la solicitud
  5. Elementos que faciliten la localización de los datos

El plazo de respuesta será de hasta 20 días hábiles, y la ejecución en los 15 días hábiles siguientes.
